# User Description

## 原始描述

> 我們公司大概有一百多個人，現在請假都是用 Line 群組跟主管說，然後主管再口頭通知 HR。很亂，常常忘記或算錯。我們想做一個系統，讓員工可以線上申請請假，主管可以在上面審核，HR 可以看到所有人的紀錄。
>
> 假別的話，我們有特休、病假、事假、喪假。特休每年一開始就給，年資不同天數不同，這個我之後可以給你 Excel。病假一年三十天，事假一年十四天，喪假看親等。這些都是法定的，應該不會差太多。
>
> 申請的時候要填開始日期、結束日期、假別、還有原因。病假要附診斷書，但目前我們也不確定是不是一定要強制上傳，有些同仁可能就是一般感冒。
>
> 主管審核就是同意或拒絕，拒絕要寫原因。如果主管超過三天沒有回，我們希望系統可以催一下，但具體怎麼催我們還沒討論好，可能是 Email，可能是系統通知。
>
> HR 的話，他們要能看到所有人的假期紀錄，還有匯出報表，但報表要長什麼樣子我們還沒有定。
>
> 對了，我們有跨部門主管的問題，有些同仁屬於兩個部門，不確定要誰來審。還有代理主管，主管出差或請假的時候誰來審，這個也還沒想清楚。

## 初步限制

- 公司規模約 100 人以上。
- 目前無任何系統，以 Line 群組溝通為主。
- 假別規則目前以法定假別為基礎，細節待 Excel 確認。
- 報表格式尚未定義。
- 跨部門主管審核規則尚未確認。
- 代理主管機制尚未確認。
