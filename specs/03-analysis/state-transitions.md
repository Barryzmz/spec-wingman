# State Transitions

## 請假申請單（LeaveApplication）狀態機

```
                    ┌─────────────┐
                    │   (新建)     │
                    └──────┬──────┘
                           │ 員工提交（FR-001）
                           ▼
                    ┌─────────────┐
               ┌───│   待審核     │───┐
               │   └──────┬──────┘   │
               │          │          │
  員工取消      │   超過3天 │ 每日中午  │ 主管審核
  （FR-007）   │   催辦通知 │ 循環發送  │ （FR-002）
               │   （FR-003）        │
               ▼          │          ├──────────────┐
        ┌──────────┐      │          │              │
        │  已取消   │      │   核准    │    拒絕（含原因）
        └──────────┘      ▼          ▼              ▼
                    ┌──────────┐ ┌──────────┐
                    │  已核准   │ │  已拒絕   │
                    └──────────┘ └──────────┘
```

### 狀態轉換明細

| State Flow ID | 實體 | 起始狀態 | 事件 | 目標狀態 | Actor | 關聯需求 |
| --- | --- | --- | --- | --- | --- | --- |
| ST-001 | LeaveApplication | （新建） | 員工填寫資訊後提交 | 待審核 | 員工 | FR-001, WR-001 |
| ST-002 | LeaveApplication | 待審核 | 主管選擇核准 | 已核准 | 主管 / 代理主管 | FR-002, WR-002 |
| ST-003 | LeaveApplication | 待審核 | 主管填寫原因後拒絕 | 已拒絕 | 主管 / 代理主管 | FR-002, WR-003 |
| ST-004 | LeaveApplication | 待審核（> 3天） | 每日中午 12:00 系統觸發 | 待審核（持續） | 系統 | FR-003, WR-004 |
| ST-005 | LeaveApplication | 待審核 | 員工執行取消 | 已取消 | 員工 | FR-007, WR-007 |
| ST-008 | LeaveApplication | （新建） | 員工提交，且無上層主管 | 已核准（自動） | 系統 | BR-010, WR-008 |

### 終止狀態

| 狀態 | 說明 |
| --- | --- |
| 已核准 | 最終狀態，不可再更改（員工取消限待審核中） |
| 已拒絕 | 最終狀態 |
| 已取消 | 最終狀態，由員工主動取消 |

## 代理主管記錄（ActingManagerRecord）狀態機

```
  主管指定代理人及期間
  （FR-006, WR-005）
         │
         ▼
  ┌─────────────┐
  │   代理生效   │  ← 代理期間內，代理人擁有審核權限
  └──────┬──────┘
         │ 代理期間結束（系統自動）
         ▼       （WR-006）
  ┌─────────────┐
  │   代理失效   │
  └─────────────┘
```

| State Flow ID | 實體 | 起始狀態 | 事件 | 目標狀態 | Actor | 關聯需求 |
| --- | --- | --- | --- | --- | --- | --- |
| ST-006 | ActingManagerRecord | （新建） | 主管指定代理人及期間 | 代理生效 | 主管 | FR-006, WR-005 |
| ST-007 | ActingManagerRecord | 代理生效 | 系統時間超過代理結束日期 | 代理失效 | 系統 | FR-006, WR-006 |
