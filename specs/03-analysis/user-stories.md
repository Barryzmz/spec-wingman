# User Stories

| Story ID | User Story | 關聯需求 | 優先級 | 狀態 |
| --- | --- | --- | --- | --- |
| US-001 | As a 員工, I want 在系統上填寫假別、日期與原因送出請假申請, so that 不需透過 Line 群組口頭請假，減少遺漏。 | FR-001, WR-001 | High | Draft |
| US-002 | As a 員工, I want 系統在我送出病假申請（>= 3 天）時提示上傳診斷書, so that 我能一次完成所有必要文件的提交。 | FR-002, BR-015, BR-016 | High | Draft |
| US-003 | As a 員工, I want 在假單尚未審核前可以撤回申請, so that 我發現填錯時可以及時修正。 | FR-005, WR-004 | High | Draft |
| US-004 | As a 員工, I want 在假單被核准或拒絕後收到 Email 與站內訊息通知, so that 我能及時知道審核結果。 | FR-010, WR-006, WR-007 | Medium | Draft |
| US-005 | As a 員工, I want 系統在額度不足時阻擋我送出申請, so that 我不會送出無效的假單。 | BR-025, FR-001 | High | Draft |
| US-006 | As a 主管, I want 在系統上查看下屬的待審核假單並同意或拒絕, so that 不需依賴口頭通知，審核過程有紀錄可追溯。 | FR-003, WR-002, WR-003 | High | Draft |
| US-007 | As a 主管, I want 拒絕假單時系統強制我填寫原因, so that 申請人能了解被拒絕的理由。 | FR-003, BR-017 | High | Draft |
| US-008 | As a 主管, I want 指定代理主管並設定代理起訖日期, so that 我不在時下屬的假單仍能被即時審核。 | FR-008, PR-004, BR-020 | Medium | Draft |
| US-009 | As a 代理主管, I want 在代理期間內審核原主管下屬的假單且結果為最終, so that 審核流程不會因主管不在而停滯。 | FR-003, FR-008, BR-026, PR-005 | Medium | Draft |
| US-010 | As a HR, I want 查看全公司所有員工的請假紀錄, so that 我能掌握整體出勤狀況。 | FR-006, PR-006 | High | Draft |
| US-011 | As a HR, I want 依部門與日期區間匯出 Excel 請假報表, so that 我能用於薪資計算與出勤統計。 | FR-007, DR-004, DR-005 | High | Draft |
| US-012 | As a HR, I want 查看哪些主管設了代理人及代理期間, so that 我能掌握審核流程的代理狀態。 | FR-009, PR-008 | Medium | Draft |
| US-013 | As a 系統, I want 在假單待審核超過 3 天後自動發送催辦通知, so that 避免假單長時間無人處理。 | FR-004, WR-005, NFR-003 | Medium | Draft |
