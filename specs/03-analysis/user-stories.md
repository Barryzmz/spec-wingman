# User Stories

| Story ID | User Story | 關聯需求 | 優先級 | 狀態 |
| --- | --- | --- | --- | --- |
| US-001 | As an 員工, I want to submit a leave application online, so that I don't need to notify my manager via Line group. | FR-001 | High | Confirmed |
| US-002 | As an 員工, I want the system to block my submission when my quota is insufficient, so that I know I've exceeded my leave entitlement before submitting. | FR-001, BR-002, BR-003 | High | Confirmed |
| US-003 | As an 員工, I want to cancel my pending leave application, so that I can update my plans without having to wait for my manager. | FR-007 | Medium | Confirmed |
| US-004 | As a 主管, I want to approve or reject leave applications in the system, so that I can manage my team's leave efficiently. | FR-002 | High | Confirmed |
| US-005 | As a 主管, I want to receive a daily reminder when an application has been pending for over 3 days, so that I don't miss applications that need my attention. | FR-003, BR-007 | High | Confirmed |
| US-006 | As a 主管, I want to designate an acting manager with a defined period, so that leave applications are still processed when I'm absent. | FR-006 | Medium | Confirmed |
| US-007 | As an HR, I want to view all employees' leave records, so that I have a complete and accurate picture of leave usage across the company. | FR-004 | High | Confirmed |
| US-008 | As an HR, I want to export leave reports in Excel format, so that I can share and analyse leave data with stakeholders. | FR-005 | High | Confirmed |
| US-009 | As an HR, I want to manage employee data including department and manager assignment, so that the system accurately reflects the organization structure. | PR-003, DR-001 | Medium | Confirmed |
