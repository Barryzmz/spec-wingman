## 核心模組說明

| 模組 | 職責 | 關聯需求 |
| --- | --- | --- |
| 請假申請模組 | 提交、取消申請；計算工作時數；驗證配額與診斷書 | FR-001, FR-007, BR-002~005 |
| 審核流程模組 | 主管審核；代理主管管理；無上層主管自動核准；催辦排程 | FR-002, FR-003, FR-006, WR-001~008, BR-006~010 |
| 通知服務模組 | 系統內通知 + Email；每日中午定時觸發催辦 | FR-003, BR-007 |
| 假期配額模組 | 計算與扣除每位員工每種假別的配額；特休從到職日起算 | DR-004, BR-001~004, DEC-017, DEC-018 |
| 假日曆模組 | 維護國定假日清單；提供工作時數計算依賴 | DR-007, DEC-020 |
| HR 模組 | 查閱全員紀錄；匯出 Excel（篩選條件 Q-017 Pending） | FR-004, FR-005, DR-006 |
| 員工資料模組 | HR 建立與維護員工、部門、主管關係 | PR-003, DR-001, DEC-010 |

## 關鍵業務規則（設計影響）

| 規則 | 設計影響 |
| --- | --- |
| 時數計算排除週末與國定假日（DEC-020） | 需查詢 holidays 表；工作時數計算需逐日判斷 |
| 代理主管有效期間（BR-009, WR-006） | 需排程任務每日檢查並失效過期代理記錄 |
| 無上層主管自動核准（BR-010, WR-008） | 提交申請時先查 manager_id，若 NULL 則直接 approved |
| 特休配額從到職日起算（DEC-017） | 需記錄到職日；每年週年日重新計算配額 |
| 申請時必填代理人（DEC-019） | leave_applications.proxy_employee_id 設為 NOT NULL |
| 催辦每日中午 12:00（BR-007） | 需定時排程 Job |
