# Glossary

| Term | 定義 | 來源 | 備註 |
| --- | --- | --- | --- |
| 特休 | 特別休假，依年資給予，每年初一次性發放 | SRC-001 | 天數對照表待 SRC-002 確認 |
| 病假 | 因疾病就醫所請之假別，一年上限 30 天 | SRC-001 | 可能需附診斷書，見 Q-003 |
| 事假 | 因私人事務所請之假別，一年上限 14 天 | SRC-001 | |
| 喪假 | 因直系或旁系親屬死亡所請之假別，天數依親等而定 | SRC-001 | 完整對照表見 Q-002 |
| 親等 | 親屬關係的遠近程度，用於決定喪假天數 | SRC-001 | |
| 診斷書 | 醫療機構出具之就醫證明文件，病假時可能需要提交 | SRC-001 | |
| 代理主管 | 主管不在時，被指定代為執行審核職責的人員 | SRC-001 | 指定機制尚未確認，見 Q-006 |
| HR | 人力資源部門，負責管理員工假期紀錄與報表 | SRC-001 | |
