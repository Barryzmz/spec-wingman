# Glossary

| Term | 定義 | 來源 | 備註 |
| --- | --- | --- | --- |
| 特休 | 年度特別休假，依員工年資給予不同天數，每年年初一次給足 | SRC-001, SRC-002 | 又稱年假、annual leave |
| 病假 | 因疾病或傷害需休養時申請的假別，年度額度 30 天 | SRC-001 | sick leave |
| 事假 | 因個人事務需處理時申請的假別，年度額度 14 天 | SRC-001 | personal leave |
| 喪假 | 因親屬過世申請的假別，天數依親等關係不同 | SRC-001, SRC-003 | bereavement leave |
| 假單 | 員工提交的一筆請假申請紀錄 | SRC-001 | leave request / leave application |
| 審核 | 主管對假單進行同意或拒絕的動作 | SRC-001 | approval / review |
| 催辦通知 | 主管超過 3 天未審核時，系統自動發送的提醒 | SRC-001 | 透過 Email 與系統通知，每日 0 點與 12 點觸發 |
| 代理主管 | 主管指定的臨時代理人，在設定期間內可代為審核假單 | SRC-001 | acting manager / deputy manager |
| 銷假 | 已核准假單的取消流程，需提交銷假單由主管審核 | SRC-001 | 明確列為第二期功能，第一期不做 |
| 親等 | 親屬關係的遠近分類，用於決定喪假天數 | SRC-003 | degree of kinship |
| 診斷書 | 醫療機構出具的證明文件，病假 3 天（含）以上時需上傳 | SRC-001 | medical certificate |
| 年資 | 員工自到職日起累計的服務時間，用於決定特休天數 | SRC-001, SRC-002 | seniority / years of service |
