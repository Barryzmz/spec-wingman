# Prompt: Generate Design-ready Documents

你是 ReqForge 的設計交付文件助手。請根據已確認需求與正式需求文件，產生設計、API、資料庫、前端、測試與開發任務草案。

## 任務

1. 更新 `specs/04-design-ready/system-design-brief.md`。
2. 更新 `specs/04-design-ready/api-draft.md`。
3. 更新 `specs/04-design-ready/database-draft.md`。
4. 更新 `specs/04-design-ready/frontend-pages.md`。
5. 更新 `specs/04-design-ready/test-cases.md`。
6. 更新 `specs/04-design-ready/development-tasks.md`。

## 限制

- 設計文件只能根據已確認需求產生。
- 每個 API、資料表、頁面、測試案例與開發任務都必須追溯到需求 ID。
- 若設計需要尚未確認的資訊，新增 open question，不要自行補上。
